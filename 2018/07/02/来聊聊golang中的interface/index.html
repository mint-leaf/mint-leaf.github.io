  • golang 中的 interface 是什么

    • interface 是接口，相信之前写过 c++，c#和 java 等语言的同学肯定不会陌生，这是在面向对象编程的过程中一直碰见的东西，以下是摘自百度百科的解释：功能是把所需成员组合起来，用来装封一定功能的集合。它好比一个模板，在其中定义了对象必须实现的成员，通过类或结构来实现它。接口不能直接实例化，即 ICount ic=new iCount() [1] 是错的。接口不能包含成员的任何代码，只定义成员本身。接口成员的具体代码由实现接口的类提供。接口使用 interface 关键字进行声明。在 golang 中，interface 也是这样的，包含了一些方法的合集。但是，对于没有泛型的 golang 来说，interface 也可以是对于泛型一种弱支持。
